# Replication Materials

This repository contains the replication materials for the article:

"Do AI-Assisted Academic Papers Receive More Citations?
Evidence from Bibliometric and NLP-Based Analyses"

## Contents
- Python analysis notebook implemented in Google Colab
- Scripts for bibliometric data processing
- NLP-based readability and text complexity analyses

## Data
The study uses openly available bibliometric data from:
- OpenAlex
- CrossRef
- Semantic Scholar
- arXiv and SSRN preprints

All data were collected in compliance with the respective data providers' terms of use.

## Reproducibility
The complete analysis pipeline is implemented in Python and can be fully reproduced by running the notebook in Google Colab.
